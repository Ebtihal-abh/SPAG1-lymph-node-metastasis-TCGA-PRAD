## Data

All data are publicly available from **UCSC Xena** (https://xenabrowser.net/datapages/).
Download the following files and place them in `data_raw/` before running the pipeline.

| File | Dataset | Version | Downloaded |
|------|---------|---------|------------|
| `TCGA.PRAD.sampleMap_HiSeqV2.gz` | TCGA PRAD → gene expression RNAseq → IlluminaHiSeq (**use TCGA hub, not GDC hub**) | 2017-10-13 | 2026-02-17 |
| `clinical-data.tsv` | TCGA PRAD → phenotype → TCGA clinical | 2019-12-06 | 2025-02-05 |
| `survival-data.tsv` | TCGA PRAD → phenotype → survival data | 2018-09-13 | 2025-02-05 |

Data sources:
- All three files were downloaded from UCSC Xena (Goldman et al., 2020)
- Survival endpoints are derived from the TCGA Pan-Cancer Clinical Data 
  Resource (Liu et al., 2018) and made available via UCSC Xena

## Execution Order

Run all scripts from the **project root** in the order listed below.

```
# Phase 1 — Data construction

08b_DE_GSEA_N1_vs_N0.R

Note: Script numbers are non-consecutive (gaps at 06, 09, 10, 14, 16)

superseded by the final pipeline.
